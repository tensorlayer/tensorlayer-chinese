为什么开发TensorDB
----------------

TensorLayer是为工业应用而开发，大规模的机器学习应用是其应用的典型场景
TensorDB是一套数据库工业，是专为应对大规模机器学习任务而开发的。


典型的应用问题如

1. 如何管理超大规模的训练数据

2. 如何使用比电脑硬盘还大的数据讯练模型

3. 如何管理不同的版本以及架构的模型

4. 如何实现模型从训练，测试到布署的自动化

TensorDB是为解决如上的问题而生，它的设计主要围绕着三个原则

1. 一切皆数据. TensorDB数据库包括构建模型的全部信息。

2. 一切读取都是查找

3. 数据可能无限大，流式处理以及按需加载
